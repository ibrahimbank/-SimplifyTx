## Installation

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Setup

1. Clone the repository:

  a. git clone https://github.com/your-username/transaction-dashboard.git
  
  b. cd transaction-dashboard

2. npm install
# or
yarn install

Open http://localhost:3000 to view in browser.


## Production Build

npm run build
# or
yarn build
